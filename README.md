# Vila Zizel 🌿

Welcome to the **Vila Zizel** repository. This is a modern React application designed for a lovely Mediterranean accommodation located in Analipsi, Messenia, Greece.

## Features ✨

- **Home**: A beautiful introduction to the villa and its surroundings.
- **Accommodation**: Detailed information about our Mediterranean rooms.
- **About**: The story behind Vila Zizel and our commitment to hospitality.
- **Contact**: Easy way for guests to reach out and inquire about bookings.
- **Weather Widget**: Real-time weather updates for Analipsi to help guests plan their stay.

## Tech Stack 🛠️

- **React 19**: Modern frontend library.
- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS 4**: For beautiful, responsive styling.
- **Lucide React**: Clean and elegant iconography.
- **Motion**: Smooth animations and transitions.

## Getting Started 🚀

### Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Local Setup

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd Vila-Zizel
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Copy the example environment file and add your Gemini API key (if needed for AI features):
   ```bash
   cp .env.example .env
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:3000` (or the next available port).

## Deployment 🌐

This project is optimized for modern hosting platforms like Vercel, Netlify, or Cloud Run.

```bash
npm run build
```

The build output will be in the `dist` directory.
