# Vila Zizel Design Assets

Reusable SVG assets for the refined Gods' Sanctuary design system.

- `greek-key-divider.svg`: low-opacity meander divider for page or print layouts.
- `laurel-seal.svg`: VZ monogram and laurel seal for favicons, social graphics, and printed collateral drafts.
- `marble-grain.svg`: subtle alabaster/marble background texture.
- `olympus-glow.svg`: obsidian and divine-gold atmospheric background overlay.

React UI icons live in `src/components/BrandIcon.tsx` so they can inherit the site's current color tokens.
